Operators

A symbol that can assign a value to variable or perform arithmitic between variables and/or values.

Variables

Containers for storing data values.

Data Type

The type of unit of an object. E.g. string.

Tags

An HTML code that defines the structure on an HTML page.

Attributes

A modifier of an HTML element type.

Bootstrap

A CSS library with helper code.

Selector

Selectors are used to select HTML elements based on their element name, id, class, attribute and more.
